About

A brief history class about me.

  • 2012-2014

    HTML, CSS and Wordpress

    Technology has always fascinated me. Back in 2012 I was touring as a musician around Europe I decided to make a website for myself and another one for the band I was playing with. That's were I began to learn HTML and CSS, but because we need it a website as fast as possible I started to search for tools to build something good as fast as possible so I found Wordpress which allowed me to make those two websites.

  • Feb 2015-2017

    The path to become a developer

    I Left Cuba exactly on feb 14 -Valentine's Day- and moved to Miami, Florida in search for a better future and a career change. I started to study Comptia A+ and Network + to get IT certifications that will allowed me to get a network admin job without a degree. I started to learn Javascript with a book until in July 2017 I was awarded with the Knight Foundation Scholarship to attend the Wyncode Academy bootcamp in Miami. That completely changed my life and taught me the skills and tools I needed to become a web developer

  • 2017-2018

    A 4 month Internship and freelance career

    After graduating from Wyncode I had a 4 month internship at Kipu Systems. I was completely exposed and involved into what it's like to work day to day as a professional web developer. I achieved deeper insight into Agile methodologies and scrum meetings, by-weekly sprints and worked on feature implementations and bugs that was even hard to solve for experienced senior devs. I learned that a dev work enviroment is very important for the successful development of someone with the role of a junior developer -coworker relationships and advice from others- and things are usually not always beautiful, easy or pleasant.

  • Present time

    Searching for a better future

    A professional good web developer should be -in my modest opinion- someone always interested in learning new exciting things, always get better and be willing to help. For me, those feelings and the will to help an ever growing community leaded me to create my Youtube channel, this website, articles, and teach. I find myself usually hanging around documentation, personal projects discovering new technologies or tools to make my work more efficient, taking advice and giving advice in return, and refactoring code.

Weather Stalker

For any weather that might appear, I got you covered.

This was a little tough small project also from the "take home" projects from FreeCodeCamp. The goal was to be able to grab the location of the user trhough the browser using the GeoLocation API and making a GET request to the OpenWeatherMap API that returns a nice JSON object with all the weather information. Some of the technologies were CSS animations, jQuery, Bootstrap 4 and HTML 5
